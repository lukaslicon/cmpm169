Innovate

How did you use the code as a jumping off point to create something truly unique and novel? Document this step thoroughly.

I pretty much changed the code completely trying to smooth out the bar movement and making everything fully based on spectrum. I had it choose the color based on spectrum an the length of the bars based on the spectrum as well. This makes it so that whatever song is uploaded there will be a visual representation based on the sound.

I also wanted to make some other form of visual representation so I did particle explosions that are randomly on the screen based on the spectrum ranges as well. The lower bass like notes will cause explosions and then they will be colored red or blue based on the spectrum.

I also made it so that it can take .MP3, .WAV, .OGG
