Innovate

How did you use the code as a jumping off point to create something truly unique and novel? Document this step thoroughly.

I implemented random colors but it started to mess up quite a bit and almost looked to strobe like. It looked like a flashing light and i tweaked values and tried to make gradual changes in colors. After messing around with the color changing I realized it would be cooler looking to change the boarder colors rather than the inside colors. I tried to make each line individual change like as if the color would be moving along the line changing through random colors. This did not work and I ultimately messed up and created a wave of changing colors where each line changed in a wave like manner. I ended up really liking this and sticking with it. I smoothed out the transitions and I really loved how it ended up looking when you interact with the diamond to move it.

I then tried to implemt this a second time. If you hit 2 on the webpage it switches to another art interaction. This one moves/rotates the same as the other one but different objects are at play. I instead ignored the borders and used my old code where I was changing the inside color only. I made it so that it chooses random colors within a blue color pallet. These colors then fade away and it looks really cool to see them create different patterns depending on how you move your mouse and the position you leave it in.
