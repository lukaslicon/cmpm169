Integrate

How did you extend the code or combine with another technique to make something new? Document this step.

I tried to copy the code of the sine/cosine link and make another formation of spheres. I did this by creating a helix version of the sine/cosine to make it look like DNA.

Innovate

How did you use the code as a jumping off point to create something truly unique and novel? Document this step thoroughly.

I tried to change the sine/cosine code at first but it was hard to create a moving representation of a double helix (DNA). As my innovation I wanted to represent a DNA strand so instead I worked from the ground up and used the sphere example to start from 1 object and build up. I built the double helix all by myself in the end and learned a lot through trying to visually represent it. Even though I did not end up getting a moving visual representation of the double helix I am still really proud of how it ended up and the interaction that you can have with the 3D DNA strand.
