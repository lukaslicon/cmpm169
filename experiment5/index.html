Imitate

What code did you start with? (You don't need to provide the code itself, but where did you start?) Document this step briefly.

The code I started with was the 2 examples from the lecture that was given for the ascii metaball art. I really wanted to do something relating to ascii art and I liked the mouse interaction so I decided to lean towards these two examples and imitate them.

Ascii Meteballs by Joan Shin and Ascii Metaballs by Wes Modes

Innovate

How did you use the code as a jumping off point to create something truly unique and novel? Document this step thoroughly.

After copying the base code and replicating the style I innovated on the visual aspects by creating a color palette that was appealing to the eye. This made it look truly unique compared to the original project. I also thought it would be interesting to create an explode and reset button so the user can have a bit more interaction. The explosion spawns a set number of particles in the middle and it creates another unique effect that the original code base could not do.
